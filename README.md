# ESP32 S3 BLE to USB HID Bridge
Мультимедійний адаптер для підключення Bluetooth-кнопок, клавіатур, мишей до Android-магнітол через Native USB (Зазвичай Bluetooth-HID стак в них не працює). Також підійде для BIOS (наприклад, якщо потрібно щоб Bluetooth клавіатура працювала як звичайна).

Опис (About)
Цей проєкт перетворює компактну плату ESP32-S3-Zero на інтелектуальний міст. Вона отримує команди від бездротових кнопок керма по протоколу BLE (Bluetooth Low Energy) і миттєво передає їх у магнітолу як стандартні команди USB-клавіатури (HID).
Основні фішки:
Native USB HID: Магнітола бачить пристрій як рідну клавіатуру, не потребує драйверів.
Низька затримка: Завдяки бібліотеці NimBLE-Arduino.
1. Залізо (Hardware)
Контролер: ESP32-S3-Zero (або будь-яка інша ESP32-S3 з Native USB).
Кнопки: Будь-які китайські BLE-кнопки для керма.
Магнітола: Будь-яка на базі Android з підтримкою OTG.
2. Налаштування Arduino IDE
Board: ESP32S3 Dev Module
USB Mode: Hardware CDC and JTAG
USB CDC On Boot: Enabled
Бібліотеки: NimBLE-Arduino
5. Інструкція з використання
Скомпілюйте та прошийте код.
Використайте вбудований сканер (у коді), щоб дізнатися MAC-адресу ваших кнопок.
Впишіть адресу в змінну targetAddress.
Підберіть коди кнопок через Serial Monitor (якщо він працює) або методом тестування.
